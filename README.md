# Project discription (项目描述)
This project is suitable for experiments on human-computer interaction technology, a limited elective course for fourth-year undergraduate students majoring in Computer Science and Technology, School of Computer Science and Technology, Shandong University. The author took this course in the fall semester of the 2025-2026 academic year.

Note⚠️: The author will reuse this website as his personal homepage after completing this course
